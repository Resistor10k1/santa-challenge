Reads the package drop locations from a CSV file.

The format of a location is: GiftId,Latitude,Longitude,Weight. The latitude and longitude can be in format 'degree' or 'radiant'.

Parameters
pathPath to the csv file containing the gift data
separatorSeparator character used in the csv file.
outputList with the data extracted as Giff objects
giftFactoryFactory which produces the Gift objects (see GiftFactory.hpp and Gift.hpp for more information.)
